boltzmann.utils.option

class boltzmann.utils.option.Some(val: T)[source]

Bases: Generic[T]

A structural pattern matchable Option type for Python.

For example

x = to_opt(x)
y = to_opt(y)
match x, y:
    case Some(x), None:
        # do something with x
    case None, Some(y):
        # do something with y
    case Some(x), Some(y):
        # do something with both
    case _:
        # both are none :shrug:

Compare this to the equivalent code without using Some:

match x, y:
    case x, None if x is not None:
        # do something with x
    case None, y if y is not None:
        # do something with y
    case x, y if x is not None and y is not None:
        # do something with both
    case _:
        # both are none :shrug:
